# Backpack/BorderSize

## Usage

### Objective-C

```objective-c
#import <Backpack/BorderSize>

CGFloat myAppBorderBorderSmall = BPKBorderSizeSm;
CGFloat myAppBorderBorderLarge = BPKBorderSizeLg;
```


### Swift

```swift
import Backpack

let smallBorder = Backpack.BPKBorderSizeSm
let largeBorder = Backpack.BPKBorderSizeLg
```
