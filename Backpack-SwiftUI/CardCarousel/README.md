## Usage

Example of a CarouselCard: 
```swift
BPKCardCarousel(
    cards: [
        BPKCarouselCard(
            content: {
                AnyView(
                    Rectangle()
                        .foregroundColor(BPKColor.skyBlue)
                )
            },
            title: "Test Title",
            description: "Test description",
            contentAccessibilityLabel: "Blue rectangle"
        ),
        BPKCarouselCard(
            content: {
                AnyView(
                    Rectangle()
                        .foregroundColor(BPKColor.skyBlue)
                )
            },
            title: "Test Title",
            description: "Test description",
            contentAccessibilityLabel: "Blue rectangle"
        ),
        BPKCarouselCard(
            content: {
                AnyView(
                    Rectangle()
                        .foregroundColor(BPKColor.skyBlue)
                )
            },
            title: "Test Title",
            description: "Test description",
            contentAccessibilityLabel: "Blue rectangle"
        )
    ],
    curentIndex: .constant(0)
)
```
