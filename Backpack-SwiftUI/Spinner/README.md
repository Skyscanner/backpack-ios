## Usage

### Basic spinner
If you don't specify a style, it will default to textPrimary

```swift
BPKSpinner(.sm)

BPKSpinner(.sm, style: .textPrimary)
BPKSpinner(.lg, style: .disabled)
BPKSpinner(.xl, style: .onDarkSurface)
```
